📞 FAQ

Q: Can I use these prompts with other LLMs (GPT-4, Gemini)?
A: Yes! The prompts are framework-agnostic. Adjust model names in prompts
   as needed. We recommend Claude for consistency.

Q: What if I only need part of this (e.g., just the dashboard)?
A: Use Section 3 from CRIME_INTEL_DESIGN_PROMPTS.md for that page's specs.
   Pick relevant Prompt Sets from TACTICAL_IMPLEMENTATION_PROMPTS.md.

Q: Can I skip dark mode?
A: Yes. It's optional. Add later if needed. Tailwind makes it easy.

Q: What if my backend API differs from the spec?
A: The API contract is a suggestion. Adjust React Query hooks (Prompt 8.1)
   to match your actual endpoints.

Q: Do I need all 5 pages for MVP?
A: Start with Dashboard + Investigation (core use cases). Add Hotspot
   Intelligence → Network Analysis → Predictive Analytics incrementally.
